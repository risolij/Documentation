## 🛠️ Local Development
If you want to run this locally or contribute a fix:

   1. Clone the repo:
   
   ``` bash
   git clone git@github.com:risolij/Documentation.git
   ```
   
   2. Install dependencies:
   
   ``` bash
   curl --proto '=https' --tlsv1.2 -sSf -L https://install.determinate.systems/nix | sh -s install
   ```
   
   3. Navigate to Project
   
   ``` bash
   cd Documentation
   ```

   4. Serve 
   ``` bash
   zensical serve
   ```
   
   
## 🧠 Philosophy
I follow the "Digital Garden" approach:

* Growth over perfection: Notes start as "seedlings" and grow over time.
* Bi-directional linking: Connecting ideas rather than just filing them in folders.
* Public by default: Sharing the process, not just the finished result.
