## Contributing Guide
If you want to run this locally or contribute a fix

### NIX: Setting up your local development
#### Clone the repo
``` bash
git clone git@github.com:risolij/Documentation.git
```

#### Install NIX 
```bash
sh <(curl -L https://nixos.org/nix/install) --daemon
```

#### Navigate to Project
``` bash
cd Documentation
```

### DOCKER: Setting up your local development   
#### Clone the repo
``` bash
git clone git@github.com:risolij/Documentation.git
```
#### Install docker
[Docker Get Started](https://www.docker.com/get-started/)
   
#### Navigate to Project
``` bash
cd Documentation
```

#### Start Container
``` bash
docker compose up
```
